Game Development Engines

There are many different types of game engines used to create games.


  • Unity

    • - Unity uses C#, JavaScript, and Boo (type of CLI language)



  • Construct 2

    • - Construct 2 is easy to learn and does not need any knowledge of coding.



  • Stencyl

    • - Stencyl uses the Haxe programming language.



  • Godot

    • - Godot uses C++ and C#.



  • Game Maker Studio 2

    • - Game Maker Studio uses their own language called Game maker Language.



  • Gamefroot

    • - Gamefroot does not require any coding language knowledge.



  • Flowlab

    • - FlowLab does not require any coding language knowledge.



  • Blender

    • - Blender requires some knowldege of Pytho 3



To enter the game developing field, it would benefit you to know; C++, JavaScript, C# for Unity, HTML5, CSS3, Java, SQL
